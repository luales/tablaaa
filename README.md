# Visualizador - Torneo Tetris (Computador 11)

Este proyecto se encarga de leer los archivos `.json` generados por los 10 computadores cliente y mostrar una tabla de posiciones en tiempo real usando Pygame.
para realizar pruebas de puntajes ejecutar generador_pruebas.py y al mismo tiempo dashboard.py
